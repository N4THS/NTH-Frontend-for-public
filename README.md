## 기업 소개 홈페이지 개발 프로젝트
해당 저장소는 프로젝트의 프론트엔드 애플리케이션 저장소입니다.
<br>
코드 공개를 위해 생성된 저장소이며, 실제 운영 환경과 다를 수 있습니다.
<br>
실제 배포 및 개발은 별도의 비공개 저장소에서 진행됩니다.
